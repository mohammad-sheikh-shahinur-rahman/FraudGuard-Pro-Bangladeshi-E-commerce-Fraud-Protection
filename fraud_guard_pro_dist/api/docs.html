FraudGuard Pro API Documentation

Integrate Bangladesh's most powerful fraud detection engine into your own platform.

GET /api/v1/check.php

Analyze Phone Number

Fetch aggregated courier statistics and risk assessment for a specific phone number.

Authentication

All requests must include a Bearer Token in the header.

Authorization: Bearer FG-SECRET-789

Parameters

Parameter Type Description
phone String 11-digit BD mobile number.

Sample Request (cURL)

curl -X GET "https://yourdomain.com/api/v1/check.php?phone=01712345678" \
     -H "Authorization: Bearer FG-SECRET-789"
