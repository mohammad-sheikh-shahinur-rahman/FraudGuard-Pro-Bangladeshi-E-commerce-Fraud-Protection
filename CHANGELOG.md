# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.1] - 2026-06-07
### Added
- Professional `composer.json` for package management.
- Detailed Installation Guide in README.
- PSR-4 Autoloading support.

### Added
- Initial release of bKash Tokenized Checkout DevHub.
- Interactive Dashboard with Chart.js analytics.
- Laravel & Raw PHP integration components.
- SQLite transaction logging.
- CSV Export & Live Search features.
