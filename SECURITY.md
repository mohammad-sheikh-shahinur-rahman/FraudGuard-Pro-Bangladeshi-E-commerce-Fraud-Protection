# Security Policy

## Supported Versions

Currently, the following versions of bKash DevHub are supported with security updates:

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0.0 | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please send an e-mail to Mohammad Sheikh Shahinur RAHMAN via [LinkedIn](https://www.linkedin.com/in/mohammad-sheikh-shahinur-rahman/). All security vulnerabilities will be promptly addressed.

Please include:
- A description of the vulnerability.
- Steps to reproduce.
- Any potential impact.

Thank you for helping keep this project secure!
